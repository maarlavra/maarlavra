### About me

I'm currently studying Information Systems while working as an Occupational Safety Technician, a field I'm also certified in along with Administration. My long-term goal is to build a career as a FullStack Developer, and I'm steadily working toward that by learning programming logic, C, Java, JavaScript, Python, and mobile development with Flutter and Dart. I also enjoy UI/UX design and have been picking up Figma along the way. Beyond development itself, I'm drawn to the security side of tech — networking, information security, cloud, pentesting, DevOps/SecOps, and data analysis are all areas I want to explore further as I grow. Python is my favorite language so far.
